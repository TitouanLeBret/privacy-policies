# Politique de Confidentialité - Eurêka

**Dernière mise à jour:** 20 Mars 2026
**Éditeur:** Titouan LE BRET

---

## 1. INTRODUCTION

Eurêka ("l'Application") respecte votre vie privée. Cette politique explique comment nous collectons, utilisons et protégeons vos données personnelles.

**Points clés:**
- ✅ Aucune donnée personnelle n'est collectée
- ✅ L'Application fonctionne entièrement en local sur votre appareil
- ✅ Vos équations mathématiques ne quittent jamais votre téléphone/Mac (sauf si vous choisissez une API externe)
- ✅ Conformément au RGPD et à la loi Informatique et Libertés

---

## 2. DONNÉES COLLECTÉES

### 2.1 Données traitées LOCALEMENT (sur votre appareil)
- ✅ **Équations mathématiques** - Capturées par caméra ou sélectionnées depuis fichiers
- ✅ **Résultats de calculs** - Générés localement, jamais envoyés à nos serveurs
- ✅ **Paramètres de préférence** - Type d'opération, modèle IA choisi (stockés localement)

**Ces données restent sur votre appareil. Elles ne sont JAMAIS transmises à tiers.**

### 2.2 Données optionnelles si vous choisissez une API externe
Si vous configurez une clé API (OpenAI, Claude, Gemini), vous choisissez d'envoyer:
- ✅ **Équation mathématique** - Pour résolution via leur service
- ❌ **Identité** - Nous ne partageons jamais qui vous êtes
- ❌ **Localisation** - Jamais collectée
- ❌ **Historique** - Vous seul avez accès à votre historique

**Ces données sont protégées:**
- Chiffrement Keychain (iOS/macOS) - clés API non accessibles
- Transmission HTTPS sécurisée
- Conformité à la politique de l'API (voir leurs conditions)

---

## 3. UTILISATION DES DONNÉES

### Sans API externe (Apple Intelligence ou calcul local)
- ✅ Aucune donnée ne quitte l'Application
- ✅ Aucun suivi, aucune analyse
- ✅ Aucune publicité

### Avec API externe (votre choix)
- Équation → Provider API (OpenAI/Claude/Gemini)
- Résultat ← Retour à l'Application
- Votre clé API: chiffrée localement, jamais partagée avec nous

**Important:** Chaque fournisseur a sa propre politique. Consultez-les:
- OpenAI: https://openai.com/privacy
- Anthropic: https://www.anthropic.com/privacy
- Google: https://policies.google.com/privacy

---

## 4. STOCKAGE DES DONNÉES

### Stockage LOCAL (sur votre appareil)
- **iOS:** Keychain + UserDefaults (protégé par chiffrement du téléphone)
- **macOS:** Keychain + UserDefaults (protégé par chiffrement du Mac)
- **Clés API:** Keychain système (chiffrement matériel)
- **Préférences:** UserDefaults (local uniquement)

### Sauvegarde iCloud
- ❌ Les clés API ne sont JAMAIS sauvegardées dans iCloud
- ❌ Les équations ne sont JAMAIS sauvegardées dans iCloud
- ✅ Seules les préférences (non sensibles) peuvent être synchronisées

### Suppression
- Vous contrôlez tout: supprimez l'app = données supprimées
- Aucune donnée n'est conservée ailleurs
- Aucune sauvegarde sur nos serveurs

---

## 5. SÉCURITÉ

Nous protégeons vos données avec:
- ✅ **Keychain chiffré** - Pour les clés API
- ✅ **HTTPS/TLS** - Pour communications API externes
- ✅ **Device encryption** - Chiffrement du téléphone/Mac
- ✅ **Pas de serveurs tiers** - Pas de cloud, pas de base de données centrale

---

## 6. COOKIES & TRACEURS

- ❌ **Aucun cookie** utilisé
- ❌ **Aucun tracker** (Google Analytics, Mixpanel, etc.)
- ❌ **Aucune publicité** ciblée
- ❌ **Aucun suivi cross-app**

## 7. DROITS DE L'UTILISATEUR (RGPD)

Vous avez droit à:

### 7.1 Droit d'accès
- Demandez: `titouanlebretsupport@gmail.com`
- Réponse: dans 30 jours

### 7.2 Droit de suppression
- Supprimez l'Application = toutes vos données supprimées
- Aucune donnée centralisée à supprimer (car tout est local)

### 7.3 Droit de portabilité
- Vos données vous appartiennent
- Vous pouvez les exporter à tout moment
- (Nous n'avons pas vos données - elles sont sur votre téléphone)

### 7.4 Droit d'opposition
- Pas de publicité ciblée = rien pour s'opposer
- Vous pouvez simplement ne pas utiliser d'API externe
